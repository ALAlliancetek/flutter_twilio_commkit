# flutter_twilio_commkit — Example

Minimal example app demonstrating the `flutter_twilio_commkit` SDK.

## Setup

1. Copy your credentials into `lib/main.dart`:
   - Replace `YOUR_VOICE_ACCESS_TOKEN` / `YOUR_VIDEO_ACCESS_TOKEN`
   - Replace `YOUR_ACCOUNT_SID`, `YOUR_API_KEY_SID`, etc.

2. For a full setup guide see:
   [`docs/integration_guide.md`](../../../docs/integration_guide.md)

## Run

```bash
flutter pub get
flutter run
```
